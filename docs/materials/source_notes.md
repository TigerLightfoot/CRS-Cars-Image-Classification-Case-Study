# Sources to Print for the Hard Copy Folder

For the CS3 hard copy, print these two sources and attach them to the right side of the manila folder.

## Source 1: CRS Cars Dataset Page

Print the main Meta-Album CRS Cars Dataset page.

Why this source matters:  
This page gives the basic dataset details students need to understand the project, including the dataset name, domain, number of classes, number of images, image size, source, and OpenML download options.

## Source 2: Datasheet for VCL.CRS Dataset

Print the CRS dataset datasheet PDF.

Why this source matters:  
The datasheet gives more detail about the dataset’s purpose, composition, collection process, preprocessing, intended uses, and limitations. This is useful because students need to understand not only how to use the data, but also what the data was originally created for.
